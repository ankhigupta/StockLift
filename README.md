# StockLift

B2B dead inventory auction marketplace.

Businesses can list excess stock for auction and buyers bid in real time.

## Tech Stack

- Mobile: React Native (iOS + Android) + React Native Web
- Backend: Node.js / Express
- Database: PostgreSQL (AWS RDS)
- Real-time: Socket.IO + Redis Pub/Sub (Upstash)
- Payments: Razorpay
- Push Notifications: Firebase FCM
